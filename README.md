# Monst_Maze 
A Java-based game developed as part of the Object-Oriented Programming course at the University of Sri Jayewardenepura. This project demonstrates the practical application of core OOP principles in a real-world software development task.  

-------------------------

## 🎮 Project Overview ##
---
This repository contains the source code for a game designed and implemented using Java. The goal of the project was to create a playable, well-structured application that emphasizes code reusability, modularity, and efficient logic.

Key Features
---
+ Game Loop: Robust enforcement of game rules and state management.  
+ User Interaction: Responsive controls and intuitive gameplay.  
+ Win/Loss Logic: Integrated conditions for game completion and scoring.  
+ Replay System: Option to restart the game without exiting the application.
